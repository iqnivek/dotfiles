## Installation

* Symlink .bashrc
* Symlink .bashrc\_rvm (if using rvm)
* Symlink .vimrc
* Symlink .gitconfig (or copy and edit)
* Symlink .jshintrc (optional)
* Install vundle: https://github.com/gmarik/Vundle.vim
